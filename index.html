⌨️ Comandos de clase

print(.) función
print(objeto) — muestra en consola.
🔑 sugerencia Separa varios valores con comas: print("A", 20). Usa sep= y end=.
print(f".) f-string
print(f"texto {variable}") — interpolación.
🔑 sugerencia Ideal para formatear: print(f"El resultado es {num:.2f}"). Desde Python 3.6.
input(.) entrada
variable = input("mensaje: ") — lee texto.
🔑 sugerencia Siempre retorna string. Convertir con int() o float() si se espera número.
float(.) conversión
float(cadena_o_entero) → número decimal.
🔑 sugerencia Útil para operaciones con decimales. Ej: altura = float(input("altura:")).
int(.) conversión
int(objeto) → entero (trunca si es float).
🔑 sugerencia int("42") → 42. Cuidado con decimales: int(3.99) → 3.
array(.) módulo array/list
from array import array; a = array('i',[1,2,3]) o usar listas [].
🔑 sugerencia Para BioPython, listas son comunes. Ej: secuencias = ["ATGC", "CGTA"].
numpy. librería
import numpy as np; arr = np.array([1,2,3]).
🔑 sugerencia Operaciones vectoriales rápidas. Para datos biológicos: np.mean(elongaciones).
math. módulo math
import math; math.log(100,10), math.sqrt(81), constantes.
🔑 sugerencia Funciones matemáticas: math.sin, cos, exp, log10. Útil para BioPython (distancia, estadística).
import sentencia
import modulo o from modulo import funcion.
🔑 sugerencia Para BioPython: from Bio import SeqIO. Organiza el código y evita colisiones.
for ciclo
for variable in iterable: (dos puntos y sangría).
🔑 sugerencia Recorrer listas, cadenas, archivos. Ej: for base in "ATCG": print(base).
